SLN Made with Visual Studio 2017, .NET Framework v4.7.2

The compiler should be located in the root folder of the game build. In the folder with the exe file there should be a txt file called dataPath.txt. In it you need to enter the name of the folder that is located in the root of the game build with _Data at the end. The game must be built using Mono, it will not work with IL2CPP
